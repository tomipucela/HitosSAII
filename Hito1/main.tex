\section{Tema y Supuesto elegido: Información sobre el clima}
El supuesto elegido fue el de información sobre el clima, teniendo como tema principal la integración de datos climáticos históricos y actuales en España.
El objetivo de este proyecto es proporcionar información climática, tanto histórica como actual, para España con el objetivo de que se pueda contrastar junto con condiciones geográficas y demográficas,
proporcionando herramientas útiles para la población en general para que puedan tomar decisiones informadas sobre sus actividades diarias, viajes, etc.

Las consultas que hemos elegido realizar para este supuesto son:
\begin{itemize}
    \item Datos de radiación ultravioleta y calidad del aire en provincias de Cataluña.
    \item Datos de temperatura media en marzo del año pasado en la estación de Aguilar de Campoo (Palencia) de la AEMET en comparación con los datos históricos de eltiempo.es y su altura según AEMET y datos.gob.es
    \item Última predicción de temperatura de la AEMET y de eltiempo.es para municipios de más de 5.000 habitantes en la provincia de Valladolid.
\end{itemize}

\section{Esquema del mediador}
\subsection{Especificacion}
El esquema mediador estará compuesto por 6 tablas: Municipio, Predicción\_UV\_Ciudad, Predicción\_Eltiempo, Predicción\_AEMET, Datos\_Historicos y Provincias.
Con este esquema mediador pretendemos integrar los datos de las diferentes fuentes (AEMET, eltiempo.es y datos.gob.es) para proporcionar una visión unificada de la información climática en España.

La estructura de las tablas es la siguiente:
\begin{itemize}
    \item Municipio(\underline{municipio}, codigo\_provincia, poblacion\_munipio, superficie, altitud)
    \item Prediccion\_UV\_Ciudad(\underline{municipio}, id, uv)
    \item Predicción\_Eltiempo(\underline{municipio}, calidad\_aire, temperatura\_minima\_media, temperatura\_maxima\_media, temperatura\_media, prediccion\_minima, prediccion\_maxima)
    \item Prediccion\_AEMET(\underline{municipio}, provincia, temperatura\_minima, temperatura\_maxima, sensacion\_termica\_maxima, sensacion\_termica\_minima)
    \item Datos\_Historicos(\underline{municipio,} \underline{fecha,} temperatura\_media, temperatura\_minima, temperatura\_maxima, altura, provincia)
    \item Provincias(\underline{codigo\_provincia}, nombre\_provincia,comunidad\_autonoma)
\end{itemize}

\subsection{Descripción de las tablas}
\begin{itemize}
    \item La tabla \textbf{Municipio} contiene información sobre los municipios de España, incluyendo su código de provincia, nombre actual, población, superficie y altitud. Su finalidad es proporcionar información geográfica y demográfica sobre los municipios para contextualizar los datos climáticos.
    \item La tabla \textbf{Prediccion\_UV\_Ciudad} almacena datos sobre la radiación ultravioleta en todas las ciudades españolas, con el objetivo de proporcionar información sobre la exposición al sol. Esta tabla solo guarda la radiación UV correspondiente a la fecha actual.
    \item La tabla \textbf{Predicción\_Eltiempo} contiene información sobre datos diarios de temperatura y predicciones para mañana para diferentes municipios, incluyendo la calidad del aire y las temperaturas mínimas, máximas y medias.
    \item La tabla \textbf{Prediccion\_AEMET} contiene las predicciones de temperatura mínima, máxima y sensación térmica para diferentes municipios según la AEMET. Esta tabla se utiliza para comparar las predicciones de la AEMET con las de eltiempo.es y proporcionar una visión más completa de las condiciones climáticas futuras.
    \item La tabla \textbf{Datos\_Historicos} almacena datos históricos de temperatura media, mínima y máxima, así como la altura y provincia de cada municipio en una fecha específica. Esta tabla se utiliza para comparar los datos actuales con los históricos y analizar tendencias climáticas a lo largo del tiempo.
    \item La tabla \textbf{Provincias} contiene información sobre las provincias de España, incluyendo su código, nombre y comunidad autónoma a la que pertenecen. Esta tabla se utiliza para relacionar los municipios con sus respectivas provincias y comunidades autónomas, facilitando la integración de datos geográficos en el análisis climático.
\end{itemize}
